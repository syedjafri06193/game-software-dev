# 🎮 Game Development Journey (Java & Python)

## 🚀 Overview

This repository documents my journey of learning game development using **Java** and **Python**—from foundational concepts to advanced systems design.

Rather than treating this as a collection of disconnected projects, this repo is structured as a **progressive exploration of game development**, where each project builds on previous knowledge and introduces new technical and design challenges.

The goal is not just to build games, but to understand:

* How game engines work under the hood
* How to design scalable and maintainable systems
* How to translate mathematical and physical concepts into interactive experiences

---

## 🧠 Philosophy

This journey is guided by a few core principles:

* **Learn by building** — Every concept is reinforced through implementation
* **Progressive complexity** — Each game introduces new systems and ideas
* **Reusability** — Core systems are abstracted into a reusable engine layer
* **Reflection** — Each project includes notes on challenges, trade-offs, and improvements

---

## 🛠️ Tech Stack Evolution

### Java Track

* Core Java (OOP, data structures)
* Java Swing / JavaFX (early rendering & UI)
* libGDX (2D/3D game development framework)
* LWJGL (low-level graphics and engine development)

### Python Track

* Python fundamentals
* Pygame (2D game development)
* AI & simulation experiments (pathfinding, behaviors)

## 📈 Progression Roadmap

### 🟢 Beginner (Fundamentals)

Focus: Game loops, input handling, basic logic

* Pong
* Snake
* Tetris

### 🟡 Intermediate (Systems & Design)

Focus: Architecture, physics, game state management

* Breakout
* Space Invaders
* 2D Platformer

### 🔴 Advanced (Engineering Depth)

Focus: Scalability, AI, networking, and engine design

* Top-Down Shooter
* Pathfinding & AI Simulation
* Multiplayer Game (client-server architecture)
* 3D Game (custom rendering / engine work)

---

## 🧩 Engine Development

Alongside the games, I am building a **modular game engine layer** to support:

* Rendering abstraction
* Input handling
* Collision detection & physics
* Scene and entity management

Over time, earlier games will be refactored to use this engine.
